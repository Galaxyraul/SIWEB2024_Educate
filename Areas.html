Cómo Calcular Áreas

Calcular el área de diferentes figuras geométricas puede ser útil en muchos contextos. Aquí te mostramos cómo calcular áreas de algunas formas comunes:

Área del Rectángulo

El área de un rectángulo se calcula multiplicando su longitud por su anchura.

Ejemplo: Longitud = 5 unidades, Anchura = 3 unidades

Área = Longitud × Anchura = 5 × 3 = 15 unidades cuadradas

Área del Círculo

El área de un círculo se calcula multiplicando π (pi) por el cuadrado del radio.

Ejemplo: Radio = 4 unidades

Área = π × Radio² = π × 4² = 16π unidades cuadradas

Área del Triángulo

El área de un triángulo se calcula multiplicando la base por la altura y dividiendo el resultado por 2.

Ejemplo: Base = 6 unidades, Altura = 8 unidades

Área = (Base × Altura) / 2 = (6 × 8) / 2 = 24 unidades cuadradas
